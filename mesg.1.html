Synopsis

mesg [y | n]

Description

The mesg utility sets or displays the messaging status of the controlling terminal.

The mesg operates on the first of standard input, standard output and standard error which is associated with terminal device.

Options

y

Allow other users to send messages to the controlling terminal.

n

Prevent other users from sending messages to the controlling terminal.

If no options are specified, mesg writes the current messaging status to standard output.

Diagnostics

The following error values are returned:

Error value: 0
Diagnostic: Successful completion, messages are permitted.
Error value: 1
Diagnostic: Successful completion, messages are denied.
Error value: >1
Diagnostic: An error occurred.

Standards

The mesg utility conforms to the Single UNIX Specification, Version 3 (SUSv3).
